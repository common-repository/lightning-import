This plugin allows users to utilize the Lightning Import data processing service to import products into woocommerce
Includes replacement search form and filter functionality to allow filters by custom criteria


== Installation ==

Install the plugin. Make sure you have registered with our free service at [https://lightningimport.com](https://lightningimport.com). Enter your username and password in the plugin settings page. 
Upload files to our service or setup your account to sync to our service and your products will be automatically imported and updated. See [https://lightningimport.com/instruction](https://lightningimport.com/instruction) for a step by step walkthrough!
